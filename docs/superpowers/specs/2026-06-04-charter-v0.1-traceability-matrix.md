# Plainweave v0.1 Requirements Traceability Matrix

## Scope

This RTM maps Plainweave planning requirements to ADRs, contracts, tests, and
implementation slices. It deliberately separates v0.1 implementation scope from
later federation and verification scope.

## Scope Legend

- **v0.1**: implement in first local-core slice.
- **v0.2+**: explicitly deferred.
- **Governance**: decision/design artifact required now, implementation later.

## v0.1 Approval Checklist

Before product code starts:

- ADR-001 through ADR-004 accepted.

- v0.1 implementation plan approved.
- Contract fixture plan approved.
- Quality gates accepted as merge criteria.
