# Plainweave v0.1 Local Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Use superpowers:test-driven-development for every code task.

**Goal:** Build Plainweave's first local requirements core: requirements, drafts, immutable versions, acceptance criteria, manual/proposed trace links, JSON CLI envelopes, and state/contract tests.

**Architecture:** Keep the v0.1 core local and deterministic. Use a small service layer over SQLite storage, with CLI commands calling the service and returning versioned JSON envelopes. Do not implement verification records, baselines, impact, MCP mutations, or federation integrations in this slice.

**Tech Stack:** Python 3.12+, SQLite stdlib, argparse stdlib, pytest, ruff, mypy, uv.

**Prerequisites:**
- ADR-001 through ADR-004 accepted.
- Review docs/superpowers/specs/2026-06-04-plainweave-interface-contract-hardening.md.
- Review docs/superpowers/specs/2026-06-04-plainweave-v0.1-traceability-matrix.md.
- Review docs/superpowers/specs/2026-06-04-plainweave-contract-fixture-plan.md.
- Review docs/superpowers/plans/2026-06-04-plainweave-v0.1-work-package-execution-guide.md.
- Confirm `make ci` passes on the scaffold before starting.

---

## Scope Guardrails

Implement only:

- `plainweave init`
- `plainweave doctor`
- local `.plainweave/plainweave.db`
- requirements table, drafts table, versions table
- acceptance criteria
- manual/proposed trace links
- append-only events for v0.1 mutations
- JSON CLI success/error/list envelopes
- CLI commands for local lifecycle and trace operations
- unit, contract, and state-machine tests

Do not implement:

- verification records
- baselines
- impact engine
- MCP server or MCP mutation tools
- Loomweave, Filigree, Wardline, or Legis integrations
- import/export
- context.md generation
- LLM assistance

## Target File Map

Create:

- `src/plainweave/errors.py` - domain exceptions and error codes.
- `src/plainweave/envelopes.py` - JSON success/list/batch/error envelope builders.
- `src/plainweave/paths.py` - project discovery and `.plainweave/` path helpers.
- `src/plainweave/store.py` - SQLite connection, migrations, and low-level persistence.
- `src/plainweave/models.py` - dataclasses/enums for requirements, versions, criteria, trace links, events.
- `src/plainweave/service.py` - domain state-machine operations.
- `src/plainweave/cli_commands.py` - argparse command registration and handlers.
- `tests/state/` - lifecycle/state-machine tests.
- `tests/contracts/` - fixture shape tests.
- `tests/fixtures/contracts/` - v0.1 JSON contract fixtures.

Modify:

- `src/plainweave/cli.py` - dispatch real v0.1 commands.
- `tests/test_cli.py` - update scaffold tests for real command behavior.
