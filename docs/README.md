## Precursor-era docs (provenance) — read with the reframe in mind

This repo was seeded from the `~/charter` precursor (local requirements/verification
core + read-only MCP, started 2026-06-04) and reframed + renamed. The documents
below were authored for **Charter, requirements-down**, before the reframe. They
were mechanically renamed into the Plainweave/Weft namespace for consistency, but
their *framing* (requirements-down, and any "five-member" / roster claims) reflects
the precursor era, **not** Plainweave's actual status. Plainweave is **gap-named
and not in the launch cutover**. Treat these as history; the canon above wins on
any disagreement.
